# Documentation

This directory contains public technical notes for Marble Love.

## Public Technical Notes

- `rom-layout.md`: expected MAME ROM files and region layout.
- `hardware-map.md`: Atari System 1 hardware notes used by the reimplementation.
- `video-system.md`: renderer and video hardware notes.
- `sound-system.md`: sound CPU/chip architecture and current implementation
  notes.
- `level-header-format.md`: verified level descriptor header format.
- `fixture-inventory.md`: large fixture inventory and public-size rationale.
- `context-map.md`: compact map for AI agents and maintainers.

## Internal Development History

Historical PRDs, AI-agent task files, handoffs, and long investigation notes live
under `docs/internal/` or `docs/archive/`. They are useful for maintainers, but
they are not the recommended starting point for new users.
